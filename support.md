# Layered by Layering 지원

## 문의하기

이메일: [layered@iwannavy.com](mailto:layered@iwannavy.com)

문의하실 때 다음 정보를 함께 보내주시면 빠른 답변이 가능합니다:

- iOS 버전 (설정 → 일반 → 정보)
- 앱 버전 (앱 내 설정에서 확인)
- 기기 모델 (예: iPhone 15 Pro)
- 문제 상황 설명 및 가능하면 스크린샷

## 자주 묻는 질문

### Q. iCloud 동기화가 안 됩니다.

다음을 순서대로 확인해주세요:

1. **iOS 설정 → Apple ID → iCloud → Layered by Layering** 스위치가 켜져 있는지
2. **네트워크 연결** 상태 (Wi-Fi 또는 셀룰러)
3. **iCloud 저장 공간**에 여유가 있는지 (설정 → Apple ID → iCloud → 저장 공간)
4. 앱을 완전히 종료 후 다시 실행

### Q. 구독을 취소하고 싶습니다.

**iOS 설정 → Apple ID → 구독**에서 "Layered by Layering"을 선택해 취소할 수 있습니다. 앱 개발자는 구독 해지를 대행할 수 없습니다.

### Q. 환불을 원합니다.

환불은 Apple App Store에서 직접 신청하실 수 있습니다: [https://support.apple.com/ko-kr/HT204084](https://support.apple.com/ko-kr/HT204084)

### Q. 캘린더 또는 미리 알림이 앱에 표시되지 않습니다.

**iOS 설정 → 개인정보 보호 및 보안 → 캘린더 / 미리 알림**에서 Layered by Layering에 권한이 허용되어 있는지 확인해주세요. 권한을 거부하신 경우 앱에서 데이터를 불러올 수 없습니다.

### Q. 앱에서 완료한 미리 알림이 Apple 미리 알림 앱에 반영되지 않습니다.

미리 알림 쓰기 권한이 필요합니다. 설정 → 개인정보 보호 → 미리 알림 → Layered by Layering을 **"전체 접근"**으로 설정해주세요.

### Q. 데이터를 다른 기기로 옮기려면 어떻게 하나요?

두 기기에서 동일한 Apple ID로 iCloud에 로그인하고, 두 기기 모두에서 iCloud 동기화를 켜시면 자동으로 동기화됩니다.

### Q. 내 데이터를 모두 삭제하려면?

1. 앱 내에서 개별 항목 삭제: 항목을 길게 눌러 삭제
2. 전체 삭제: 앱을 삭제한 후 **설정 → Apple ID → iCloud → 저장 공간 관리 → Layered by Layering**에서 iCloud 데이터를 삭제
